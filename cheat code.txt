pwd - periksa kedudukan pengguna/anda
git init - setup git repository
git config --global user.name "nama anda" -> konfigur nama
git config --global user.email "nama anda" -> konfigur email
git status - periksa status fail dalam folder repositry/projek
git add "nama fail" -> masukkan fail dalam stage
git commit -m "masukkan mesej" -> bagi tahu apa perubahan yang dibuat.
git branch "nama branch" -> buat cawangan/branch baru
git checkout "nama branch" -> pergi ke direktori branch
git rm --cached "nama fail" -> untuk keluar dari stage
git add --all/A -> add semua fail ke dalam stage
git status --short -> periksa perubahan fail yang memaparkan maklumat ringkas
git log - untuk perika status perubahan dalam fail.
